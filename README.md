# kimi-k2-6-tensorrt-llm-benchmark
OpenAI-compatible benchmarking suite for evaluating inference performance using TensorRT-LLM serving of moonshotai/Kimi-K2.6. Designed for the inference-frameworks assignment on 8× NVIDIA RTX PRO 6000 Blackwell GPUs, supporting tensor-parallel inference, 4-bit quantization, and long-context workloads.
